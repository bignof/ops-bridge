命名空间

命名空间 = agent（一台一个，code = agentId，1:1）
🔑
每个 agent 持本命名空间的 pull-token（worker 拉插件经本机 agent，worker 自身零凭据）。pull-token 轮换即时生效、show-once。

服务

逻辑服务 = 插件策略落点；实例自动发现，服务一键纳管
🔁
服务这层只关心「装哪些插件、什么版本」，目录 / 镜像 / nacos 由实例自动发现。常规走「从发现一键纳管」（零敲）；「手动预建」仅用于首次上线前预置插件集（绕开鸡生蛋）。服务不纯自动建——纳管 + 配插件是显式动作。
